Tutorial de HTML

Introducción

HTML es un lenguaje markup estándar para páginas web. Su sigla proviene de Hyper Text Markup Language.

Primeros comandos

Nuevo archivo HTML

Si necesitamos generar un nuevo archivo .html podemos crear un archivo "nombre.html" y dentro de él podemos utilizar el comando "!+enter" que nos genera una plantilla de html 

            
                <!DOCTYPE html>
                <html lang="es">
                <head>
                    <meta charset="UTF-8">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    <title>Document</title>
                </head>
                <body>
                
                </body>
                </html>

Imágenes, audios y videos

Hay muchos atributos diferentes que podemos controlar, pero los básicos para agregar imagen, audio o video a nuestra página web son estos. 

            
                Imágenes: <img src="nombre.png" width="200" height="300" alt="esta es una imagen">
            

            
                Audio: <audio src="nombre.mp3">
            

            
                Video: <video src="nombre.mp4">

Metatags

La etiqueta es <meta>, y sus elementos representan tipos de metadatos que no pueden expresarse usando los elementos "title", "base", "link", "style" o "script". Ponemos estos metadatos en el <head>. Son "self-closing". 

            
                <meta name="description" content="Esta es mi página web.">
Todo este tipo de metadatos serán leídos por el browser. Por ejemplo, un metadato útil es aquel que mantiene la proporción de la página cuando se cambia de dispositivo. Este metadato es 
            
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
o cuántos segundos tenés que esperar para que el browser se actualice. 
            
                <meta http-equiv="refresh" content="30">
